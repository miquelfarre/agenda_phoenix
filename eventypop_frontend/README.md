# EventyPop Frontend

Public web application for EventyPop - discover events, follow organizations, and manage subscriptions.

## Stack

- **Next.js 15** with App Router
- **TypeScript**
- **Shadcn/ui** for UI components
- **Tailwind CSS** for styling
- **TanStack Query** for API calls
- **Zustand** for state management

## Getting Started

```bash
# Install dependencies
npm install

# Run development server (port 3003)
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Features

- Event discovery
- Organization profiles
- Calendar subscriptions via share codes
- Event search and filtering
- User authentication
- Personalized event feed

## Adding Shadcn Components

```bash
npx shadcn@latest add button
npx shadcn@latest add card
npx shadcn@latest add dialog
```

Open [http://localhost:3003](http://localhost:3003) to view the application.

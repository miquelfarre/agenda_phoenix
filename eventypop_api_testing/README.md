# EventyPop API Testing

API testing tool for EventyPop - test endpoints, manage multiple users, and verify integration flows.

## Stack

- **Next.js 15** with App Router
- **TypeScript**
- **Tailwind CSS** for styling (no UI library - custom components)
- **TanStack Query** for API calls
- **Zustand** for state management

## Getting Started

```bash
# Install dependencies
npm install

# Run development server (port 3002)
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Features

- Multi-user testing (switch between users)
- API endpoint explorer
- Request/response inspection
- JSON viewer
- Integration test flows
- Real-time API monitoring

## Usage

1. Select a test user (private or public)
2. Choose an endpoint to test
3. Configure request parameters
4. Send request and view response
5. Verify expected behavior

Open [http://localhost:3002](http://localhost:3002) to access the testing tool.
